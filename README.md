# 🛒 DNS Markdown Bot (Java 21)

Бот для мониторинга уценки DNS. Следит за ценами, пишет историю изменений и присылает уведомления.

## 🛠 Стек
* **Java 21**
* **Spring Boot 4**
* **PostgreSQL**
* **Playwright**
* **Telegram Bot SDK**
* **Prometheus & Grafana**

## ⚡ Что я реализовал:
* **История цен**: Любое изменение цены фиксируется. Бот присылает уведомление с зачеркнутой старой ценой(согласно фильтрам пользователя).
* **Архивация**: Товары, исчезнувшие из продажи, автоматически переносятся в архив.
* **Оптимизация БД**: Настроил батчинг (сохранение пачками по 50 записей) через Sequence-генераторы.
* **Авторизация**: Список разрешенных пользователей кэшируется в памяти для мгновенной проверки доступа.
* **Отчеты**: Выгрузка актуальной базы товаров в CSV-файл прямо в чат.
* **Мониторинг**: Интеграция с Micrometer, Prometheus и Grafana. Мониторинг состояния JVM, пула соединений HikariCP и метрик парсинга в реальном времени.

## 🚀 Запуск
### 1. Склонируйте репозиторий

```bash
git clone https://github.com/vgribv/DNSParser
```
### 2. Настройте переменные окружения
* Создайте файл .env и добавьте туда переменные: `BOT_TOKEN`, `BOT_NAME`, `DB_URL`, `DB_USERNAME`, `DB_PASSWORD`, `PROXY_HOST`, `PROXY_PORT`, `GRAFANA_PASSWORD`.
### 3. Запустите стек

```bash
docker-compose up -d --build
```
